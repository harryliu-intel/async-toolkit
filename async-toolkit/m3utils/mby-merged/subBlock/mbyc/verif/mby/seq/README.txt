
 this directory include all MBY sequences
 
 <mby_training_seqlib.sv> include the initial training that need to enable the MBY.
 
 <mby_traffic_seqlib.sv> include all the MBY configuration sequences.
 
 <mby_traffic_seqlib.sv> include the MBY traffic (TX DMA and RX DMA) sequences

 *Note:* _You must run first the training seq followed by config sequences before enable MBY traffic_

To run the doc tool:
firefox $IP_ROOT/validation/testbench/seqlib/mby/docs/index.html
