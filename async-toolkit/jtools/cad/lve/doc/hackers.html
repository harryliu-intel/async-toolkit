LVE Hackers Guide

Introduction

This page attempts to provide an in depth documentation about the generated files, depandancies and quirks about the LVE. It also serves as a detailed documentation of the system itself. We start of by looking into each of the directories created by the LVE and explain/document what is contained in the files found therein.

Cell Directory

.cellname
The first line indicates the CAST CDL name potentially resubtyped (for partial extraction). The second line indicates the fully qualified cellname minus (FQCNMINUS) for partial extraction, third indicates the fully qualified cellname (FQCN); this does not include the minus part during partial extraction. The last line is the gds2 encoded cellname. This file is created before any attempt is made to run any tasks related to the specified cell.
