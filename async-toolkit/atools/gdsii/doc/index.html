Documentation of GDSII Package

Usage: aaggds [-i] [-t <topcell>] [gds_file_list]
   -i        : cell names are case insensitive
   -t <name> : specify name of top cell
   file list is a space separated list of names.
   aaggds is interactive, you will receive an aaggds< prompt

Usage: rdgds [-t topcell] gds_file < rdg_file
    rdgds parses a gds file to make a text file
    if topcell specified, only shows the single cell

Usage: wrgds rdg_file gds_file
   writes gds_file parsing rdg_file generated by rdgds

Usage: scalegds [-f] infile.gds outfile.gds
    -f  : forces overwrite of outfile
    scales infile.gds by factor of 10 larger
